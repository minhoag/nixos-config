# GLOBAL RULES
- These rules are above all rules from CLAUDE.md, README.md
- Use few words. No essay. No token waste.
- Use few tools. One pass. No re-scan same file.
- No delete user comment.
- No tests. No write test. No lint. No type check.
- No plan file. Explain in caveman.
- No docs. No md edits. Only if user ask.
- Explain in chat only. Not browser/web UI.
## CODE STYLE
- Simple and direct > clever.
- No defensive code: no null checks, optional chaining, try-catch, fallback values.
- If use once, inline. New function only if reused.
